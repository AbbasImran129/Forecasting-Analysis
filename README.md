# 📊 Hotel Resort Sales Forecasting Analysis

This project focuses on forecasting hotel resort sales to understand **future revenue trends and seasonal demand patterns**.  
The analysis was performed using **Microsoft Excel**, leveraging forecasting functions, time-series techniques, and visualization to generate actionable business insights.

---

## 📌 Project Objective

The primary objective of this project is to:  
- Predict future sales using historical data  
- Identify **trend and seasonality patterns**  
- Support decision-making in **pricing, staffing, and inventory planning**

---

## ⚙️ Tools Used

- **Microsoft Excel**
  - Data Cleaning and Structuring  
  - Forecasting Functions (`FORECAST.LINEAR`, `FORECAST.ETS`)  
  - Confidence Interval Calculation (`FORECAST.ETS.CONFINT`)  
  - Charts and Visualization  

---

## 🧭 Process Overview

### **Step 1 — Data Preparation**
- Organized historical sales data (2021–2024)  
- Ensured consistency in date formats and values  
- Structured dataset for time-series analysis  

### **Step 2 — Trend Forecasting**
- Applied `FORECAST.LINEAR` to identify overall growth trends  
- Established baseline projections  

### **Step 3 — Seasonal Forecasting**
- Used `FORECAST.ETS` (Exponential Smoothing)  
- Captured seasonal demand patterns  
- Identified peak and low-demand periods  

### **Step 4 — Confidence Interval Estimation**
- Calculated using `FORECAST.ETS.CONFINT`  
- Determined upper and lower prediction bounds  

### **Step 5 — Visualization**
- Created charts comparing:
  - Actual vs Forecasted Sales  
  - Linear vs Seasonal Forecast  
  - Confidence Intervals  

---

## 📊 Key Insights

- Sales show a **consistent upward trend**  
- Strong **seasonal spikes in Q3 & Q4**  
- Linear forecasting alone is insufficient  
- Confidence intervals highlight prediction uncertainty  
- Seasonal forecasting provides more reliable insights
